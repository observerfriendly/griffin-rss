## Workflow Quick-Start

### Winner outreach
1. Pull winner CSV — verify 161 entries, check no-form flags
2. Build YAMM sheet: merge fields | test row | unsubscribe column
3. Draft template — badge + ceremony + exhibition
4. Test send one row, verify rendering
5. Stage full send — confirm before firing

### Exhibition tracker update
1. Filter by form status — flag new no-form winners
2. Update file status per latest Drive uploads
3. Flag missing items — verify each row, do not assume completion
4. Surface gaps before building outbound comms

### Board communication
1. Check tdc-board.slack.com for current context
2. Verify current roster — don't use stale member list
3. Draft in neutral governance tone
4. Stage for Griff review before sending
